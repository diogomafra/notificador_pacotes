Notificador de pacotes (Correios)
=================================

Este projeto é uma extensão do Google Chrome que notifica quando há alterações no status de entrega de um pacote nos correios.

Criei esse projeto apenas para testes e para aprender como funcionam as extensões do Chrome.

Para maiores informações a respeito das extensões do Chrome, visite esse link: http://code.google.com/chrome/extensions/getstarted.html
